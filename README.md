# workable-converter
基于libreoffice实现的文档转换工具模块，正在开发中，暂勿使用

## 技术栈

* LibreOffice:v6.2.3

* jodconverter:4.2.2

* cglib动态代理 + 懒汉工厂模式 + 策略模式 + 装饰器模式

* qtools-property管理配置文件(application.yml、bootstrap.yml、workable-converter.yml三种命名的配置文件任意包含一种即可)

## 功能

* doc、docx、html转PDF

* doc、docx转每一页的png

* pdf转每一页的png

* 多个png按照顺序合并为一个PDF文件
